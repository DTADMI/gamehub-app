# Credits for External Assets

This project may include or load external assets (images, sounds, music) that are licensed under permissive terms.
Please add attribution details here for any assets that require credit (e.g., CC‑BY). CC0 assets (e.g., most Kenney
packs) do not require attribution, but you may still list them for completeness.

## How to add a credit

- Title: Asset name or pack name
- Author: Creator name (and/or handle)
- Source URL: Direct link to the asset or collection page
- License: e.g., CC‑BY 4.0 (include link to the license text)
- Notes: Any modifications you made (e.g., trimming, normalization)
